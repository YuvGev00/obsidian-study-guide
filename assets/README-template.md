# <Course Name> — Study Vault

An interlinked Obsidian vault generated from the course material: atomic **term**
and **formula** notes, full teaching-walkthrough **lecture** notes, **MOC**
indexes, diagrams/animations, and spaced-repetition flashcards.

## How to use

1. Open this folder as a vault in [Obsidian](https://obsidian.md) (File → Open vault → this folder).
2. Start at **[[Home]]** for the map; or **[[All Lectures]]** to read in course order.
3. Hover any `[[link]]` to preview the note (hover previews are pre-enabled).
4. Open **Graph view** to see how concepts connect.

## Recommended community plugins (optional)

Install from Settings → Community plugins. The vault works without them; these
just light up extra features:

- **Spaced Repetition** (`st3v3nmw`) — turns the `## Review` cards in each term
  note into a study queue. Cards are deck-tagged by lecture (`#flashcards/…`).
- **Dataview** (`blacksmithgu`) — powers the live tables in the index notes
  (without it, those blocks show as plain code; the static link lists still work).

Flashcards also export to Anki via `Obsidian_to_Anki` / Yanki if you prefer Anki.
